Encryption App Launcher

Note: The project has been reorganized into separate frontend and backend folders. You need to start both services to use the application.

Important: Make sure MongoDB is running before starting the backend server.

Starting the Application

Follow these steps to start the application:

1. Start the Backend Server

Open a terminal and run:

cd backend
npm install
npm run dev

This will start the backend server at http://localhost:5000.

2. Start the Frontend Development Server

Open another terminal and run:

cd frontend
npm install
npm run dev

This will start the frontend development server at http://localhost:5173.

3. Access the Application

Once both servers are running, you can access the application at:

Open Application

Project Information

This is a full-stack web application for secure encryption and decryption of text and files, featuring user authentication and RSA key pair management.

Features

  • User authentication (login/register)
  • Text encryption/decryption
  • File encryption/decryption
  • RSA key pair generation and management
  • Dark/light theme toggle
  • Responsive design

For more information, please refer to the README.md file in the project root.
